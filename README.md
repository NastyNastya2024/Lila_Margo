# Meta Morfeya — Сайт проводника по игре Лила

Авторский сайт Маргариты (Meta Morfeya) — проводника трансформации через игру Лила и древние практики.

## Разделы сайта

- **О проводнике** — рассказ о Маргарите, место для фотографий
- **Услуги** — Игра Лила (групповая и индивидуальная), практики для души и Рода
- **Игра Лила** — полноценная онлайн-версия игры с 72 клетками
- **Контакты** — связь для записи

## Технологии

- Чистый HTML, CSS, JavaScript
- Без бэкенда — всё на фронтенде
- Готово для размещения на GitHub Pages

## Размещение на GitHub Pages

1. Загрузите репозиторий на GitHub
2. Settings → Pages → Source: Deploy from a branch
3. Branch: main, folder: / (root)
4. Сайт будет доступен по адресу `https://username.github.io/Lila_Margo/`

## Добавление фотографий

Замените блоки `.photo-placeholder` на теги `<img>`:

```html
<img src="images/photo1.jpg" alt="Описание" class="photo-img">
```

Создайте папку `images/` и поместите туда фотографии.

## Локальный запуск

```bash
python3 -m http.server 8080
```

Откройте http://localhost:8080
